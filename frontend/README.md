# Crop Guidance Frontend

A modern React + Vite frontend for the DCF-SEL Crop Guidance System.

## Setup

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Environment Variables

Create a `.env` file:

```
VITE_API_URL=http://localhost:8000
```

## Build

```bash
npm run build
```

Built files will be in `dist/`.
